1 MySQL SELECT - избира информация, която ще се изведе

1 MySQL WHERE - това е клауза, която проверява дали условие е правилно

1 MySQL AND, OR, NOT - това са логически операции използвани за where-a

0 MySQL ORDER BY - подрежда колони (редове)

1 MySQL INSERT INTO - добавя запис в таблица

1 MySQL NULL Values - това е стойност, която се слага като поле няма стойност

1 MySQL UPDATE - променя информация на даден ред

1 MySQL DELETE - изтрива ред
1 MySQL LIMIT - показва колко редове ще се изведат от select

0 MySQL MIN and MAX - избира минимална или максимална стойност от таблица (от колона)

0 MySQL COUNT, AVG, SUM - брои, намира средното, сумира и резултата е нова колона (реално връща резултата, не създава нова колона)

MySQL LIKE - сравнява

1 MySQL Wildcards - това са символи, които се използват за заместване на символ/и
  _ - един символ
  % - 0 до много

1 MySQL IN - дали е в дадено множество

1 MySQL BETWEEN - дали е в даден диапазон

1 MySQL Aliases - това са наименования, които се използват за колони, таблици за по-лесно работа

1 MySQL Joins - съюзява две таблици

0 MySQL INNER JOIN - не знам
0 MySQL LEFT JOIN - съюзява се към лявата страна (много е бегло, взима всички стойности отляво и добавя съвпадащите отдясно)
0 MySQL RIGHT JOIN - съюзява се към дясната страна (-||-||-)
0 MySQL CROSS JOIN - не сме учили
0 MySQL Self Join - не сме учили
0 MySQL UNION - не сме учили
1 MySQL DISTINCT - означава, че се връщат само уникалните стойности
0 MySQL GROUP BY - групират се нещата на база дадена колона (при min,max,AVG,sum,count)

1 MySQL Database - означава се че това е датабаза

1 MySQL Create DB - създава се датабаза

1 MySQL Drop DB - ако съществува се изтрива

1 MySQL Create Table - създава таблица в датабаза

0 MySQL Drop Table - изтрива датабаза

0 MySQL Alter Table - това се използва за да се създаде нова таблица, със стойностите на друга, за да може да се променя

0 MySQL Constraints -

1 MySQL Not Null - при инициализация тази колона не може да приема NULL

1 MySQL Unique - при създаване трябва да е уникална стойност

0 MySQL Primary Key - това е основен ключ, който се използва за да означи, подреждат редовете (ключ за индексиране на записите)

1 MySQL Foreign Key - когато един ключ е свързан с ключ в друга таблица

1 MySQL Check - проверка за данни, при създаването на полето

1 MySQL Default - ако не е дадена стойност, се задава означенията в default

1 MySQL Auto Increment - автоматично се запълва(обикновено при id number се дава за стойност следващото число)
