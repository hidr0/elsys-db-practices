1 MySQL SELECT
  Избира данни от таблицата/таблиците или цялата таблица, като извежда резултата в таблица.

0 MySQL WHERE
  Филтрира данни, според зададени от нас аргументи. (аргументи - ?)

1 MySQL AND, OR, NOT
  Логически оператори, които работят с where, and и or се изпълняват, когато условието, което сме задали е true,а not-false

1 MySQL ORDER BY
  Сортира таблицата във възходящ или низходящ ред, според зададени от нас условия

1 MySQL INSERT INTO
  Вкарва нови данни в таблицата

0 MySQL NULL Values
  Данни без стойност (просто полето е празно)

1 MySQL UPDATE
  Променя съществуващи данни в таблицата

1 MySQL DELETE
  Трие съществуващи данни от таблицата

0 MySQL LIMIT
  Не знам

0 MySQL MIN and MAX
  Изкарва минималната или максималната стойност от таблицата (от дадена колона)

0 MySQL COUNT, AVG, SUM
  count-брои редовете в таблицата (брои редове от дадена колона)
  avg-средна стойност от данните (средната стойност на редовете от дадена колона)
  sum-сумата на данните (сумата на редовете от дадена колона)

0 MySQL LIKE
  Прави проверка на условие, зададено от нас (сравнява стрингове)

0 MySQL Wildcards
  Не знам

0 MySQL IN
  Работи с where и изкарва само данните, които са в диапазона, зададен от нас (множество от стойности, зададени от нас)

0 MySQL BETWEEN
  Работи с where и изкарва само данните, които са в определено разстояние, зададено от нас (това е да са в определен от нас диапазон)

1 MySQL Aliases
  Наименува колона от таблицата за удобствие и по-добра четимост.

1 MySQL Joins
  В общо значение, те съединяват две или повече таблици

0 MySQL INNER JOIN

0 MySQL LEFT JOIN
  Стойностите от лявата таблица, които съвпадат с тези от дясната, се съединяват (точно обратното, взима всичко отляво и добавя съвпадащите отдясно)

0 MySQL RIGHT JOIN
  Стойностите от дясната таблица, които съвпадат с тези от лявата, се съединяват (обратното)

0 MySQL CROSS JOIN
  Не знам

0 MySQL Self Join
  Не знам

0 MySQL UNION
  Не знам

0 MySQL DISTINCT
  Не знам

0 MySQL GROUP BY
  Извежда само редове от таблици, които съвпадат, според условие, зададено с нас (групира записите по някакво зададено условие)
