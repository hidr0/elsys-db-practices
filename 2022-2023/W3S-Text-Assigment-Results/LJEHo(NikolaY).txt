0 MySQL SELECT- избира от коя таблица взимаме инфромацията (база данни)
1 MySQL WHERE- слага условие за това какво от избраната таблица да се вземе
1 MySQL AND, OR, NOT and- и (може да се добави нещо), or- или (трябва да е вярно едно или друго нещо), not- не (да не отговаря на някакво условие или просто да не е нещо)   /малоумен отговор но се схваща идеята/
1 MySQL ORDER BY- нарежда това което вадим от таблицата (по колони), по дефолт е във възходящ ред,
1 MySQL INSERT INTO- добавяме информация в таблицата (ред)
1 MySQL NULL Values- стойности, който са един вид празни (стойности, който нямат стойност)
1 MySQL UPDATE- променяме добавена инофрмация (ред) (не само ред)
1 MySQL DELETE- изтрива информация (ред) (не само ред)
1 MySQL LIMIT- слага лимит колко реда да се извадят от таблицата
1 MySQL MIN and MAX мин- изважда най-малката стойност от подадена му колона, макс- вади най-голяма колона от подадена му коона
1 MySQL COUNT, AVG, SUM  COUNT- брой елементите в подадена му колона, AVG- предполагам връща средно аритметичната стойност на подадената колона, SUM- оново предполагам, че връща сбора на стойностите на подадената колона
1 MySQL LIKE- работи за стрингове и проверява дали нещо се съдържа в стринг
0 MySQL Wildcards- не сме го учили
1 MySQL IN позволява повече условия в WHERE или съкращава неща с OR
1 MySQL BETWEEN избира елемент в даден range, между някакви стойности
0 MySQL Aliases променя име на колона в таблицата (не е подробно)
1 MySQL Joins- слива 2 таблица чрез някаква обща колона
0 MySQL INNER JOIN ми добавя се втората таблица нз (кво ми се отваряш бе кринге)
0 MySQL LEFT JOIN втората колона се добавя от ляво (не си подробен)
0 MySQL RIGHT JOIN втората колона се добавя от дясно (не си подробен)
0 MySQL CROSS JOIN добавя се само общото от втората колона в първата? (добавя всичко)
1 MySQL Self Join ми таблицата се добавя със себе си?
0 MySQL UNION обединява два SELECT-та (два или повече)
1 MySQL DISTINCT като unique за queuery-то
0 MySQL GROUP BY групира информацията по колона (по какъв начин)

0 MySQL Database избира база данни
1 MySQL Create DB създава база дании
1 MySQL Drop DB дропва базата
1 MySQL Create Table създава таблица
0 MySQL Drop Table дропва таблица (изтрива)
1 MySQL Alter Table промени по таблицата
0 MySQL Constraints не знам
1 MySQL Not Null задава че трябва да е различно от Null
1 MySQL Unique неможе 2 такива елемента да са с еднаква стойност
1 MySQL Primary Key главен ключ, главно id на редовете на таблицата
1 MySQL Foreign Key чужд ключ, от друга таблица и се използа за да съединят 2 таблици
1 MySQL Check проверява дали отговаря на някакво условие
1 MySQL Default ако при добавяне на елемент не му дадена друга стойност Default задава някаква дефолтна
0 MySQL Auto Increment че сам си задава стойности, използва се за Primary Key и всеки нов ред ще има нова стойност различна от предишната (увеличават се с едно)
