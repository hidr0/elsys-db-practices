1 MySQL ORDER BY - подрежда по дадена колона
1 MySQL INSERT INTO - добавя нов запис в таблицата
1 MySQL NULL Values - празна стойност
1 MySQL UPDATE - актуализира даден запис
1 MySQL DELETE - изтрива даден запис
1 MySQL LIMIT - дава се някакъв лимит за редове които да се покажат
1 MySQL MIN and MAX - намира минималната и максималната стойност на даденото поле като минава през всички записи
1 MySQL COUNT, AVG, SUM - брой, средно аритметично и сума на дадено поле
1 MySQL LIKE - сравнение на стринг
0 MySQL Wildcards - не знам
1 MySQL IN - проверява дали стойността на полето е в изброените стойности
1 MySQL BETWEEN - проверява дали стойност на полето е между две други 
0 MySQL Aliases - не знам
0 MySQL Joins - джойнва 2 таблици (това е вярно , но може малко повече обяснение)
1 MySQL INNER JOIN - джойнва общите записки на 2 таблици( на това и долните 3 ще дам точка, но искам малко специфициране кои данни събираме и от къде ги взимаме ;) )
1 MySQL LEFT JOIN - джойнва всички записки от ляво със съвпадащите от дясно
1 MySQL RIGHT JOIN -  джойнва всички записки от дясно със съвпадащите от ляво
1 MySQL CROSS JOIN - джойнва всички записки от ляво и от дясно
0 MySQL Self Join - не знам
1 MySQL UNION - обединява 2 таблици
0 MySQL DISTINCT - не знам
0 MySQL GROUP BY - групира по колони( затворих си очите горе тук няма да мога, вярно но искам малко повече обяснения)   

1 MySQL Database - база данни

1 MySQL Drop DB - унищожава база данни

1 MySQL Drop Table - унищожава таблица
1 MySQL Alter Table -  променя таблицата
0 MySQL Constraints - не знам

0  MySQL Unique - уникално( уникално какво??? тази клауза работи върху цели колони и казва дали всички стойности в нея са уникални)
0 MySQL Primary Key - уникален ключ( това е вярно , но уникалният ключ се намира някъде, в таблица, не е достатачно специфицирано)
0 MySQL Foreign Key - свързва две таблици( неговата цел е да свърже 2 таблици, но не го прави директоно , а като в себе си съхранва Primary key-а на другата таблица) 
1 MySQL Check - проверява дали стойност е в колона
1 MySQL Default - стойност по подразбиране
1 MySQL Auto Increment - инкрементира се автоматично( какво се инкрементира, това е клауза която позволява някаква член данна да се увеличи с 1, но мисля че си разбрал за какво говориш)
