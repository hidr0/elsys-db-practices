1 MySQL SELECT- избира таблицата, която трябва да покаже
1 MySQL WHERE-  това е условие, което филтрира по някакъв начин таблицата
1 MySQL AND, OR, NOT- това са бинарни операции и, или и не
1 MySQL ORDER BY- подрежда таблицата по някакъв специфичен начин
1 INSERT INTO- слага данни в таблицата, редове
1 MySQL NULL Values- това са данни, които отначало нямат никаква стойност
1 MySQL UPDATE- променя редове от таблицата 
1 MySQL DELETE- изтрива редове от таблицата
0 MySQL LIMIT- лимитира колко реда да се покажат от таблицата
1 MySQL MIN and MAX- взимат най-малката или голямата стойност от някоя колона в таблицата
1 MySQL COUNT, AVG, SUM- COUNT- брои, AVG- казва средно аритметичната стойност, SUM- сумира   всичко това в дадена колона
1 MySQL LIKE- проверява дали стойностите който му даваш са точно тези, които трябва, пр. точно 10 цифри или да има @ някъде
0 MySQL Wildcards- нещо лудо :) (използват се за замяна на един или повече символа в стринг)
0 MySQL IN- помага за where да станат нещата по-лесно (проверява дали стойността на полето е в изброените стойности)
0 MySQL BETWEEN-нз (проверява дали стойност на полето е между две други)
0 MySQL Aliases- нз (дава временно име на таблица или колона от таблица)
1 MySQL Joins- обединява таблици
0 MySQL INNER JOIN- нз
1 MySQL LEFT JOIN- обединява се като първата таблица е отляво, а втората се залепя отдясно
1 MySQL RIGHT JOIN-  обединява се като първата таблица е отдясно, а втората се залепя отляво   (май)
0 MySQL CROSS JOIN- нз
1 MySQL Self Join- обединява се със себе си (ми нз)
1 MySQL UNION- залепя 2 резултата (каза го в час:)
0 MySQL DISTINCT- същото като unique(променливата която е дистинкт трябва да има различна стойност при всяка своя прочетена инстанция)
1 MySQL GROUP BY- групира слепените таблици по дадено нещо
 
0 MySQL Database- казваш коя database искаш да използваш
1 MySQL Create DB- правиш си база
0 MySQL Drop DB- дропва база (знам, че е тъпо, но не измислих нищо по-умно)
1 MySQL Create Table- създава се таблица
0 MySQL Drop Table- дропва таблица
1 MySQL Alter Table- дава да се променят колони
0 MySQL Constraints-  ми нз(използва се за специфициране на правила за таблицата)
0 MySQL Not Null- казва се че някой параметър от таблицата не трябва да е 0
1 MySQL Unique- казва се че някой параметър трябва да е уникален в цялата колона
1 MySQL Primary Key- това е ид то на таблицата 
0 MySQL Foreign Key- това трябва за да може да се обединят 2 таблици по общ ключ (поле или сбор от полета от таблицата които реферират Primary Key на друга таблица )
1 MySQL Check- проверява за правилни данни още при инсъртването
1 MySQL Default- слага си дефолтно нещо
1 MySQL Auto Increment- автоматично си генерира
