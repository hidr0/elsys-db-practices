1 MySQL SELECT - Избира колони от таблица или всички със символа "*" и после с FROM се избира и съответната таблица за визуализиране 
1 MySQL WHERE - Поставя се кондиция на някаква колона или SELECT query
1 MySQL AND, OR, NOT - Логически операции, които се използват с кондиционните keyword-ове. AND проверява дали два израза връщат True, OR проверява дали поне един от изразите връща True, NOT връща обратното на израза, ако е True връща False, ако е False - True (не го прочетох цялото ма изглежда правилно)
1 MySQL ORDER BY - Подрежда таблицата по някаква колона като може да се специфицира дали е в низходящ или възходящ ред
1 MySQL INSERT INTO - Вкарва ред в таблицата
1 MySQL NULL Values - Това може да е стойност на всичко, без значение от това какъв тип е. То не значи 0, а просто "нищо".(няма стойност)
1 MySQL UPDATE - Обновява някаква таблица като добавя колона
1 MySQL DELETE - Трие някаква колона от таблица
1 MySQL LIMIT - Дава лимит на колко реда да се визуализират от SELECT
1 MySQL MIN and MAX - Връщат съответно най-ниската и най-високата стойност
1 MySQL COUNT, AVG, SUM - Връщат съответно броя на редове от колона, средната стойност и  сумата
1 MySQL LIKE - проверява за съответствие/подобоие на стойности
1 MySQL Wildcards - Символи в кондиции, където се проверяват VARCHAR-ове, които служат за някакъв шаблон за кондицията, тоест могат да се проверяват подобни VARCHAR-ове
1 MySQL IN - Проверява дали стойността на всяка клетка от съответна колона, дали съвпада с масива от стойности, който е подаден
1 MySQL BETWEEN - проверява дали стойността от колона е в диапазон от подадените
1 MySQL Aliases - Чрез keyword as могат да се задават други имена на колони, но оригиналното име не се променя
1 MySQL INNER JOIN - Съединяват само общото от две колони
1 MySQL LEFT JOIN - Присъединява всичко към първата таблица
1 MySQL RIGHT JOIN - Присъединява всичко към втората таблица
0 MySQL CROSS JOIN - не сме учили
0 MySQL Self Join - не сме учили
0 MySQL UNION - не сме учили
0 MySQL DISTINCT - отстранява повторения на стойности при изобразяване (само уникални)
1 MySQL GROUP BY - Групира таблицата по някаква колона

1 MySQL Create DB - Създава датабаза само ако не съществува някоя със същото име
1 MySQL Drop DB - Изтрива датабаза само ако съществува
1 MySQL Create Table - Създава таблица
1 MySQL Drop Table - Изтрива таблица
1 MySQL Alter Table - Променя колони на таблицата
1 MySQL Not Null - Казва, че в колоната не може да има стойност, която е NULL
1 MySQL Unique - Казва, че не могат да се повтарят стойности в колоната
1 MySQL Primary Key - Обозначава се главен ключ на таблицата, който се използва за индексиране
1 MySQL Foreign Key - Обозначава чужд ключ, това се използва като създаваме релации между таблици
1 MySQL Check - Проверява дали стойностите от колоната съответстват на някаква кондиция
1 MySQL Default - Задава стойност, която да се слага на клетката от съответната колона, ако не бъде специфицирана такава при създаване
1 MySQL Auto Increment - Автоматично след всяко INSERT-ване вкарва в реда колоната с този constraint с число, което  е с 1 по-високо от предишния ред
