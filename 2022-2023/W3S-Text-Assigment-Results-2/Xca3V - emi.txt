1 MySQL SELECT
Избира данни от таблицата/таблиците или цялата таблица, като извежда резултата в таблица.

1 MySQL WHERE
Филтрира данни, според зададени от нас аргументи.

0 MySQL AND, OR, NOT
Логически оператори, които работят с where, and и or се изпълняват, когато условието, което сме задали е true,а not-false (не е точно - or връща true, когато поне едно условие е true)

1 MySQL ORDER BY
Сортира таблицата във възходящ или низходящ ред, според зададени от нас условия

1 MySQL INSERT INTO
Вкарва нови данни в таблицата

1 MySQL NULL Values
Данни без стойност

1 MySQL UPDATE
Променя съществуващи данни в таблицата

1 MySQL DELETE
Трие съществуващи данни от таблицата

0 MySQL LIMIT
Не знам

0 MySQL MIN and MAX
Изкарва минималната или максималната стойност от таблицата (от колона)

0 MySQL COUNT, AVG, SUM
count-брои редовете в таблицата
avg-средна стойност от данните
sum-сумата на данните (от колона)

1 MySQL LIKE
Прави проверка на условие, зададено от нас

0 MySQL Wildcards
Не знам

0 MySQL IN
Работи с where и изкарва само данните, които са в диапазона, зададен от нас (множество, а не диапазон)

0 MySQL BETWEEN
Работи с where и изкарва само данните, които са в определено разстояние, зададено от нас (не е окей да се каже разстояние - по-добре диапазон)

1 MySQL Aliases
Наименува колона от таблицата за удобствие и по-добра четимост.

1 MySQL Joins
В общо значение, те съединяват две или повече таблици

0 MySQL INNER JOIN

0 MySQL LEFT JOIN
Стойностите от лявата таблица, които съвпадат с тези от дясната, се съединяват (цялата лява таблица, към която се присъединяват съвпадащите стойности от дясната)

0 MySQL RIGHT JOIN
Стойностите от дясната таблица, които съвпадат с тези от лявата, се съединяват (цялата дясна таблица, към която се присъединяват съвпадащите стойности от лявата)

0 MySQL CROSS JOIN
Не знам

0 MySQL Self Join
Не знам

0 MySQL UNION
Не знам

0 MySQL DISTINCT
Не знам
