MySQL SELECT- изкарва информация, някакви данни от базата.
MySQL WHERE-изкарва заявки при дадено условие
MySQL AND, OR, NOT- и, или или не- това са логически операции
MySQL ORDER BY-сортира по дадена колона
MySQL INSERT INTO-добавя нов ред в таблицата
MySQL NULL Values-някое от полетата няма стойност, не е 0 а е нищо.
MySQL UPDATE- обновява информацията в в даден ред
MySQL DELETE- изтрива редове от базата данни
MySQL LIMIT-казва се колко реда/записа да се върнат
MySQL MIN and MAX- да се върне минималната или максималната стойност о дадена колона
MySQL COUNT, AVG, SUM- брои записи при дадено условие, като създава нова колона която ни връща и винаги трябва да се ползва с group by, AVG- връща средно аритметично на дадена колона, SUM- връща сумата от дадена колона.
MySQL LIKE -използва се при сравняване.
MySQL Wildcards- не знам дали сме го учили, но не знам какво е
MySQL IN-не сме го учили/не съм го ползвала
MySQL BETWEEN-не сме го учили/не съм го ползвала
MySQL Aliases-  не знам дали сме го учили, но не знам какво е
MySQL Joins- използват се за съединяване на 2 таблици
MySQL INNER JOIN- не знам дали сме го учили ама не съм го ползвала
MySQL LEFT JOIN- хваща дясната таблица и я залепя за лявата. Изкарва всички записи от лявата таблица и тези които сме задали от дясната.
MySQL RIGHT JOIN-обратното на left join-хваща лявата таблица и я залепя за дясната. Изкарва всички записи от дясната таблица и тези които сме задали от лявата.
MySQL CROSS JOIN- не сме го учили
MySQL Self Join- не сме го учили
MySQL UNION-не сме го учили (залепва два резултата от query-та)
MySQL DISTINCT- връща само различни записи
MySQL GROUP BY- групираме по дадено условие

MySQL Database.
MySQL Create DB-създаваме база данни. Винаги трябва да проверяваме дали тя вече не съществува и ако е така да я drop-нем преди да създадем
MySQL Drop DB- изтрива базата
MySQL Create Table-създаваме таблица
MySQL Drop Table-изтриваме таблица
MySQL Alter Table-добавяне на
MySQL Constraints- не знам
MySQL Not Null- задава се стойността на дадено поле да не може да е празна, винаги трябва да има стойност
MySQL Unique- стойността зададена на някое поле трябва да е уникална
MySQL Primary Key- уникален ключ на всеки запис
MySQL Foreign Key- ключ, чрез който се join-ват таблици
MySQL Check- проверява за някакво условие
MySQL Default- слага състояние на полето, което ще е ако изрично не го променим
MySQL Auto Increment- автоматично се увеличава
