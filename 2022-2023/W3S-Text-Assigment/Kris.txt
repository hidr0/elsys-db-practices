MySQL SELECT - Избира колони от таблица или всички със символа "*" и после с FROM се избира и съответната таблица за визуализиране
MySQL WHERE - Поставя се кондиция на някаква колона или SELECT query
MySQL AND, OR, NOT - Логически операции, които се използват с кондиционните keyword-ове. AND проверява дали два израза връщат True, OR проверява дали поне един от изразите връща True, NOT връща обратното на израза, ако е True връща False, ако е False - True
MySQL ORDER BY - Подрежда таблицата по някаква колона като може да се специфицира дали е в низходящ или възходящ ред
MySQL INSERT INTO - Вкарва ред в таблицата
MySQL NULL Values - Това може да е стойност на всичко, без значение от това какъв тип е. То не значи 0, а просто "нищо".
MySQL UPDATE - Обновява някаква таблица като добавя колона
MySQL DELETE - Трие някаква колона от таблица
MySQL LIMIT - Дава лимит на колко реда да се визуализират от SELECT
MySQL MIN and MAX - Връщат съответно най-ниската и най-високата стойност
MySQL COUNT, AVG, SUM - Връщат съответно броя на редове от колона, средната стойност и  сумата
MySQL LIKE - проверява за съответствие/подобоие на стойности
MySQL Wildcards - Символи в кондиции, където се проверяват VARCHAR-ове, които служат за някакъв шаблон за кондицията, тоест могат да се проверяват подобни VARCHAR-ове
MySQL IN - Проверява дали стойността на всяка клетка от съответна колона, дали съвпада с масива от стойности, който е подаден
MySQL BETWEEN - проверява дали стойността от колона е в диапазон от подадените
MySQL Aliases - Чрез keyword as могат да се задават други имена на колони, но оригиналното име не се променя
MySQL INNER JOIN - Съединяват само общото от две колони
MySQL LEFT JOIN - Присъединява всичко към първата таблица
MySQL RIGHT JOIN - Присъединява всичко към втората таблица
MySQL CROSS JOIN - не сме учили
MySQL Self Join - не сме учили
MySQL UNION - не сме учили
MySQL DISTINCT - отстранява повторения на стойности при изобразяване
MySQL GROUP BY - Групира таблицата по някаква колона

MySQL Create DB - Създава датабаза само ако не съществува някоя със същото име
MySQL Drop DB - Изтрива датабаза само ако съществува
MySQL Create Table - Създава таблица
MySQL Drop Table - Изтрива таблица
MySQL Alter Table - Променя колони на таблицата
MySQL Not Null - Казва, че в колоната не може да има стойност, която е NULL
MySQL Unique - Казва, че не могат да се повтарят стойности в колоната
MySQL Primary Key - Обозначава се главен ключ на таблицата, който се използва за индексиране
MySQL Foreign Key - Обозначава чужд ключ, това се използва като създаваме релации между таблици
MySQL Check - Проверява дали стойностите от колоната съответстват на някаква кондиция
MySQL Default - Задава стойност, която да се слага на клетката от съответната колона, ако не бъде специфицирана такава при създаване
MySQL Auto Increment - Автоматично след всяко INSERT-ване вкарва в реда колоната с този constraint с число, което  е с 1 по-високо от предишния ред
