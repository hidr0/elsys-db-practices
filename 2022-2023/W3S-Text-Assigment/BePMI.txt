MySQL SELECT - избира информация, която ще се изведе

MySQL WHERE - това е клауза, която проверява дали условие е правилно

MySQL AND, OR, NOT - това са логически операции използвани за where-a 

MySQL ORDER BY - подрежда колони

MySQL INSERT INTO - добавя запис в таблица

MySQL NULL Values - това е стойност, която се слага като поле няма стойност

MySQL UPDATE - променя информация на даден ред

MySQL DELETE - изтрива ред
MySQL LIMIT - показва колко редове ще се изведат от select

MySQL MIN and MAX - избира минимална или максимална стойност от таблица

MySQL COUNT, AVG, SUM - брои, намира средното, сумира и резултата е нова колона

MySQL LIKE - сравнява

MySQL Wildcards - това са символи, които се използват за заместване на символ/и
_ - един символ
% - 0 до много

MySQL IN - дали е в дадено множество

MySQL BETWEEN - дали е в даден диапазон

MySQL Aliases - това са наименования, които се използват за колони, таблици за по-лесно работа

MySQL Joins - съюзява две таблици

MySQL INNER JOIN - не знам
MySQL LEFT JOIN - съюзява се към лявата страна
MySQL RIGHT JOIN - съюзява се към дясната страна
MySQL CROSS JOIN - не сме учили
MySQL Self Join - не сме учили
MySQL UNION - не сме учили
MySQL DISTINCT - означава, че се връщат само уникалните стойности
MySQL GROUP BY - групират се нещата на база дадена колона (при min,max,AVG,sum,count)

MySQL Database - означава се че това е датабаза

MySQL Create DB - създава се датабаза

MySQL Drop DB - ако съществува се изтрива

MySQL Create Table - създава таблица в датабаза

MySQL Drop Table - изтрива датабаза

MySQL Alter Table - това се използва за да се създаде нова таблица, със стойностите на друга, за да може да се променя

MySQL Constraints - 

MySQL Not Null - при инициализация тази колона не може да приема NULL

MySQL Unique - при създаване трябва да е уникална стойност

MySQL Primary Key - това е основен ключ, който се използва за да означи, подреждат редовете 

MySQL Foreign Key - когато един ключ е свързан с ключ в друга таблица

MySQL Check - проверка за данни, при създаването на полето

MySQL Default - ако не е дадена стойност, се задава означенията в default

MySQL Auto Increment - автоматично се запълва(обикновено при id number се дава за стойност следващото число)
