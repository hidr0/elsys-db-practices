MySQL SELECT- избира от коя таблица взимаме инфромацията
MySQL WHERE- слага условие за това какво от избраната таблица да се вземе
MySQL AND, OR, NOT and- и (може да се добави нещо), or- или (трябва да е вярно едно или друго нещо), not- не (да не отговаря на някакво условие или просто да не е нещо)   /малоумен отговор но се схваща идеята/
MySQL ORDER BY- нарежда това което вадим от таблицата (по колони), по дефолт е във възходящ ред,
MySQL INSERT INTO- добавяме информация в таблицата (ред)
MySQL NULL Values- стойности, който са един вид празни (стойности, който нямат стойност)
MySQL UPDATE- променяме добавена инофрмация (ред)
MySQL DELETE- изтрива информация (ред)
MySQL LIMIT- слага лимит колко реда да се извадят от таблицата
MySQL MIN and MAX мин- изважда най-малката стойност от подадена му колона, макс- вади най-голяма колона от подадена му коона
MySQL COUNT, AVG, SUM  COUNT- брой елементите в подадена му колона, AVG- предполагам връща средно аритметичната стойност на подадената колона, SUM- оново предполагам, че връща сбора на стойностите на подадената колона
MySQL LIKE- работи за стрингове и проверява дали нещо се съдържа в стринг
MySQL Wildcards- не сме го учили
MySQL IN позволява повече условия в WHERE или съкращава неща с OR
MySQL BETWEEN избира елемент в даден range, между някакви стойности
MySQL Aliases променя име на колона в таблицата
MySQL Joins- слива 2 таблица чрез някаква обща колона
MySQL INNER JOIN ми добавя се втората таблица нз
MySQL LEFT JOIN втората колона се добавя от ляво
MySQL RIGHT JOIN втората колона се добавя от дясно
MySQL CROSS JOIN добавя се само общото от втората колона в първата?
MySQL Self Join ми таблицата се добавя със себе си?
MySQL UNION обединява два SELECT-та
MySQL DISTINCT като unique за queuery-то
MySQL GROUP BY групира информацията по колона

MySQL Database избира база данни
MySQL Create DB създава база дании
MySQL Drop DB дропва базата
MySQL Create Table създава таблица
MySQL Drop Table дропва таблица
MySQL Alter Table промени по таблицата
MySQL Constraints не знам
MySQL Not Null задава че трябва да е различно от Null
MySQL Unique неможе 2 такива елемента да са с еднаква стойност
MySQL Primary Key главен ключ, главно id на редовете на таблицата
MySQL Foreign Key чужд ключ, от друга таблица и се използа за да съединят 2 таблици
MySQL Check проверява дали отговаря на някакво условие
MySQL Default ако при добавяне на елемент не му дадена друга стойност Default задава някаква дефолтна
MySQL Auto Increment че сам си задава стойности, използва се за Primary Key и всеки нов ред ще има нова стойност различна от предишната
