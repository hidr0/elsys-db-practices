MySQL SELECT - извежда таблица, която сме избрали или определени колони от нея
MySQL WHERE - филтрираме данни
MySQL AND, OR, NOT - логически операции, които се използват в where и in statement-и
MySQL ORDER BY - организираме и подреждаме колони по определен начин
MySQL INSERT INTO - вкарваме данни в таблицата като запълваме цели редове
MySQL NULL Values - това са несъществуващи неща
MySQL UPDATE - обновява редове и колони
MySQL DELETE - изтрива редове и колони
MySQL LIMIT - ограничава данните в определен диапазон
MySQL MIN and MAX - изкарва най-малката или респективно най-голямата стойност от определена колона
MySQL COUNT, AVG, SUM - операции с данни в колони(брои, средно аритметично, сумира)
MySQL LIKE - проверка подобно "if"
MySQL Wildcards - 
MySQL IN - разширява where statement като добавя повече колони
MySQL BETWEEN - избира стойност в определен диапазон
MySQL Aliases - 
MySQL Joins - съединява две таблици в select statement
MySQL INNER JOIN - съединява
MySQL LEFT JOIN - прилепва таблица отляво към друга таблица
MySQL RIGHT JOIN - прилепва таблица към друга таблица отдясно
MySQL CROSS JOIN - 
MySQL Self Join - добавя таблица към себе си
MySQL UNION - обединява две таблици
MySQL DISTINCT - 
MySQL GROUP BY - 

MySQL Database - база данни
MySQL Create DB - създава нова база, трябва да се специфира името й
MySQL Drop DB - изтрива база, отново с нейното име
MySQL Create Table - създава таблица, трябва да се поясни името й и колоните и типовете им
MySQL Drop Table - изтрива таблица с името й
MySQL Alter Table - променя данни(колони, клетки) от таблицата(добавя, изтрива, променя)
MySQL Constraints - 
MySQL Not Null - забранява на определена колона да съдържа несъществуващи данни
MySQL Unique - прави определена колона да съдържа неповтарящи се стойности
MySQL Primary Key - идентификация на ред в таблицата
MySQL Foreign Key - връзка с друга таблица и нейния primary key
MySQL Check - проверява за нещо
MySQL Default - задава стойност по подразбиране на определена колона
MySQL Auto Increment - увеличава стойност на дадена колона всеки път, когато се добави нов ред
