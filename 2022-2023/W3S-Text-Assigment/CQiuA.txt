Знания

MySQL SELECT - избираме кои колони да се покажат от дадена таблица
MySQL WHERE - условие, по което да се показват дадени записи от таблицата
MySQL AND, OR, NOT - логически операции, които се използват при WHERE
MySQL ORDER BY - по коя колона да се сортират данните
MySQL INSERT INTO - да се добави запис в дадена таблица
MySQL NULL Values - нямат стойност
MySQL UPDATE - update на запис
MySQL DELETE - delete а запис
MySQL LIMIT - лимитиране на колко записа да се покажат
MySQL MIN and MAX - показват минимална/максимална от дадена колона
MySQL COUNT, AVG, SUM - изпозва се за броене, средноаритметично, сума на стойности от колона
MySQL LIKE - изпозва се за сравнение на стрингове
MySQL Wildcards - използва се в LIKE, за да се намират подобни стрингове. Например %
MySQL IN - проверка дали стойности се намират някъде (таблица)
MySQL BETWEEN - между две стойности
MySQL Aliases - временно наименование на колона
MySQL Joins - присъединяване на таблица по някаква обща/matching колона
MySQL INNER JOIN - изкарва matching стойности
MySQL LEFT JOIN - изкарва всичко от лявата и matching стойности от дясната
MySQL RIGHT JOIN - изкарва всичко от дясната и matching стойности от лявата
MySQL CROSS JOIN - изкарва всички стойности и от двете
MySQL Self Join - предполагам таблица е join-ата със себе си  
MySQL UNION - не сме го учили
MySQL DISTINCT - същото като UNIQUE ама за query
MySQL GROUP BY - групира по дадена колона

MySQL Database - датабаза, в която се добавят таблици
MySQL Create DB - създаване на датабаза
MySQL Drop DB - да се махне датабаза  
MySQL Create Table - създаване на таблица
MySQL Drop Table - да се махне таблица
MySQL Alter Table - промяна на таблица
MySQL Constraints - ограничения
MySQL Not Null - не-празни стойности. Изпозва се при WHERE
MySQL Unique - стойност да е уникална/да не се повтаря
MySQL Primary Key - уникален ключ в дадена таблица (ID)
MySQL Foreign Key - връзка към PRIMARY KEY на друга таблица
MySQL Check - проверява за стойности
MySQL Default - ако не се подаде стойност при добавяне на запис, каква да бъде тази стойност
MySQL Auto Increment - със всяко добавяне автоматично да се увеличава с едно. Използва се при PRIMARY KEY
