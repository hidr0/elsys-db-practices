MySQL SELECT - Извежда данните от таблица.
MySQL WHERE - Филтрира данните.
MySQL AND, OR, NOT - Логически операции.
MySQL ORDER BY - Подрежда данните по дадена клауза. MySQL INSERT INTO - Вкарва данни в таблица
MySQL NULL Values - Полета без стойност
MySQL UPDATE - Променя някой ред в таблица
MySQL DELETE - Трие ред в таблицата
MySQL LIMIT - Лимитира изкараните редове при SELECT
MySQL MIN and MAX - Изкарват най-голямата или най-малката стойност
MySQL COUNT, AVG, SUM - Брои, взима средно аритметична стойност или събира дадени полета или колони, често се използват заедно с GROUP BY
MySQL LIKE - Използва се сравняване на стрингове
MySQL Wildcards - Това са специални символи използвани при LIKE, които служат за да не се задават винаги специфични символи. % се използва, за да изрази 0, 1 или много символи, а – се използва, за да означи 1 символ
MySQL IN - Проверява дали дадено число се намира в подаденото множество MySQL BETWEEN - Проверява дали дадено число или дата се намира в даден range от числа или дати
MySQL Aliases - Подава се, за да се промени името на дадена колона
MySQL Joins - Залепя таблица до текущата
MySQL INNER JOIN - не знам
MySQL LEFT JOIN - Показва данните от лявата таблица и съвпадащите данни от дясната
MySQL RIGHT JOIN - Показва данните от дясната таблица и съвпадащите данни от лявата
MySQL CROSS JOIN - не знам
MySQL Self Join - не знам
MySQL UNION - не знам
MySQL DISTINCT - Показва само уникалните полета
MySQL GROUP BY - Групира по една или повече колони
MySQL Create DB - Създава датабаза
MySQL Drop DB - Трие датабаза
MySQL Create Table - Създава таблица
MySQL Drop Table - Трие таблица
MySQL Alter Table - Променя колона на таблица
MySQL Constraints - Ограничения за данните на дадена колона MySQL Not Null - Дадено поле не може да е Null
MySQL Unique - Не може да има повторения в дадена колона MySQL Primary Key - Главния ключ на дадена таблица MySQL Foreign Key - Друг ключ на таблицата
MySQL Check - Подават се ограничения за дадена колона MySQL Default - Подават се данни по подразбиране
MySQL Auto Increment - Към дадена колона се добавя 1 при всеки нов запис
